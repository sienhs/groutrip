# 시연 영상 촬영 시나리오 (Shooting Script)

**프로젝트** 그룹 여행 협업 플랫폼 (EnjoyTrip)
**목표 길이** 본편 약 **5분** (+ 2분 숏컷 버전 별도 제공)
**핵심 메시지** "의사결정 → 일정/이동 → 정산"을 한 흐름으로 + **실시간(SSE) 협업**

> 💡 이 영상의 하이라이트는 **두 화면을 나란히 띄워 실시간 동기화**를 보여주는 것이다.
> 멤버 A가 무언가 하면 멤버 B 화면이 1초 안에 갱신되는 장면을 2~3번 의도적으로 노출한다.

---

## 0. 사전 준비 (촬영 전 반드시 세팅)

### 0-1. 계정 / 창 배치
| 항목 | 내용 |
| --- | --- |
| 창 A (왼쪽) | **계정 A = Owner**. 메인 진행자 |
| 창 B (오른쪽) | **계정 B = 멤버**. 일반/시크릿 창 또는 다른 브라우저 |
| 배치 | 두 창을 좌우로 50:50. 화면 녹화는 **전체 화면** 1개로(두 창 동시에 보이게) |
| 해상도 | 1920×1080, 브라우저 줌 100%, 모바일 뷰 폭이면 더 보기 좋음 |

### 0-2. 데이터 시드 (시간 절약 위해 미리 만들 것 / 라이브로 보여줄 것 구분)
| 미리 준비(시드) | 라이브로 시연 |
| --- | --- |
| 계정 A·B 로그인 완료 상태(로그인 장면만 짧게 재현) | 그룹 생성, 초대·참여, 장소·일정·투표·정산 |
| 계정 A 성향 설문 완료(결과 화면용) | 설문 응답 일부만 빠르게 |
| 보관함에 장소 2~3개 미리(검색 실패 리스크 줄이기) | 검색 1건은 라이브, 나머지는 미리 |
| 정산 수취정보(토스 링크/계좌) 미리 등록 | 송금 딥링크 클릭 |

### 0-3. 녹화 환경 체크
- [ ] 외부 API 정상(검색·길찾기 1회씩 사전 테스트) — 네트워크 끊김 대비
- [ ] 알림(OS/메신저) 끄기, 북마크바 정리, 시크릿창 확장프로그램 off
- [ ] 마이크 테스트(나레이션 녹음 or 자막), BGM 볼륨 낮게
- [ ] 실패 대비: 각 씬을 **개별 클립**으로 찍고 편집에서 합치면 NG 부담↓

## 2. 씬별 세부 시나리오 (초 단위 + 나레이션)

> 표기: ⏱ 머무는 시간 · 🖱 동작 · 🎙 나레이션(또는 자막) · ➡ 다음 전환

### 인트로 (0:00–0:08)
- ⏱ 8초 정지 타이틀 카드(또는 홈 화면 블러 배경)
- 🎙 "친구들과 떠나는 여행, **정하고·짜고·정산**하는 걸 한 곳에서. 그룹 여행 협업 플랫폼입니다."
- ➡ 페이드 인 → 로그인 화면

### S1. 로그인 (0:08–0:25)
- ⏱ `/login` 3초 노출 — 카카오/구글 버튼 보이기
- 🖱 **카카오로 로그인** 클릭 → (카카오 동의 화면 1초 스킵) → 홈 `/` 진입
- ⏱ 홈 6초 — 진행 중/예정 그룹, 상단 인사말
- 🎙 "소셜 로그인 한 번이면 끝. 별도 회원가입이 없습니다."
- ➡ 상단/하단 탭에서 **성향 설문** 진입

### S2. 성향 설문 (0:25–0:55)
- 🖱 `/survey` 진입 — 12문항 중 **3~4개만** 빠르게 클릭(편집으로 가속해도 됨)
- ⏱ 문항 화면 12초
- 🖱 제출 → `/survey/result` 레이더 차트 등장
- ⏱ 결과 12초 — "액티비티 80% 형" 같은 한 줄 설명 클로즈업
- 🎙 "5가지 축으로 내 여행 성향을 분석합니다. 이게 나중에 **그룹 궁합**으로 이어져요."
- ➡ 홈 → 그룹 만들기

### S3. 그룹 생성 + 초대 (0:55–1:30) ★ 실시간 1차
- 🖱 (창A) `/groups/new` — 제목 "제주도 여행", 목적지(시·도→시군구 드릴다운), 기간 선택, 커버 선택
- ⏱ 생성 폼 10초
- 🖱 **만들기** → `/groups/:id` 상세 진입, **초대코드 복사** 버튼 클릭
- 🎙 "그룹을 만들면 6자리 초대코드가 자동 생성됩니다."
- 🖱 (창B) 초대코드 입력 `/join/:code` → 참여
- ⏱ **두 창 동시 노출 5초** — 창A에 "OOO님이 합류했어요" **실시간 토스트** 뜨는 순간 강조
- 🎙 "멤버가 들어오면 **다른 사람 화면에 1초 안에** 반영됩니다. SSE 실시간 동기화예요."
- ➡ 멤버/성향 영역으로 스크롤

### S4. 그룹 성향 매칭 (1:30–1:45)
- 🖱 그룹 상세의 **멤버/페르소나** 영역 — 멤버 평균 성향 + **일치율 N%**
- ⏱ 12초 — 가장 갈리는 차원 강조 문구 클로즈업
- 🎙 "두 사람 성향을 합쳐 **그룹 궁합과 충돌 포인트**를 보여줍니다."
- ➡ 장소 탭

### S5. 장소 검색 + 보관함 + AI 요약 (1:45–2:20)
- 🖱 장소 탭 → 검색창에 "제주 흑돼지" 입력(라이브 검색 1건)
- ⏱ 결과 리스트 8초 — 사진·평점·가격대, 카테고리 칩
- 🖱 한 곳 **보관함 추가**(메모/카테고리)
- 🖱 장소 상세에서 **리뷰 AI 요약** 버튼 클릭 → 요약 텍스트 등장
- ⏱ AI 요약 10초
- 🎙 "모든 장소는 Google에서 한 소스로 검색하고, **리뷰는 AI가 요약**해줍니다."
- ➡ 일정 탭

### S6. 일정 빌더 + 이동 + 지도 (2:20–3:10) ★ 실시간 2차(선택)
- 🖱 일정 추가 — 보관함 장소를 1일차에 2~3개 배치(시간/메모)
- ⏱ 일정 리스트 10초
- 🖱 **드래그로 순서 변경** → 순서·시작시간 자동 반영
- 🖱 일정 사이 **이동 카드**의 자동차 탭 — 소요시간·거리·톨비·연료비
- ⏱ 이동 정보 10초
- (선택) 창B에서 일정 하나 추가 → 창A 실시간 반영 5초
- 🖱 `/groups/:id/map` **일정 지도** — 일자별 핀 + 이동 경로선 + 종합 이동 거리/시간
- ⏱ 지도 12초 — 핀 클릭 시 상세
- 🎙 "일정을 짜면 **카카오 모빌리티 실측 이동비용**이 붙고, 지도로 동선을 한눈에 봅니다."
- ➡ 투표(빈 일정) 진입

### S7. 일정 투표 (3:10–3:50) ★ 실시간 3차(핵심)
- 🖱 장소 미정 **빈 일정**에서 "후보 추가" → 후보 2곳 등록(보관함/검색)
- ⏱ 후보 화면 8초
- 🖱 **(창A)·(창B) 각각 1~5점 투표** — 두 창 동시 노출
- ⏱ **실시간 막대 그래프가 양쪽에서 같이 차오르는** 장면 8초 강조
- 🎙 "의견이 갈리면 투표로. **실명·실시간**으로 결과가 즉시 반영됩니다."
- 🖱 (창A=Owner) **마감** → 최다 득표 후보가 정식 일정으로 자동 승격
- ⏱ 승격 결과 6초
- ➡ 숙소

### S8. 숙소 선정/예약 (3:50–4:10)
- 🖱 `/groups/:id/plan` 또는 숙소 영역 — 숙소 선정 → 예약가 입력 / 예약 사진 업로드
- ⏱ 14초 — SELECTED → BOOKED 상태, 숙박 기간 표시
- 🎙 "숙소도 정하고 예약가를 넣으면 **숙박비가 정산에 자동으로** 들어갑니다."
- ➡ 정산 탭

### S9. 비용 정산 (4:10–4:50) ★ 차별점 클라이맥스
- 🖱 정산 탭 — 지출 등록 1건(금액/결제자/카테고리, 분담 균등 또는 비율)
- ⏱ 지출 등록 8초
- 🖱 일정/숙소의 **"정산에 추가"** → 이동·숙박비 자동 지출 등록
- 🖱 **정산 매트릭스** — "A → B 30,000원" 같은 **최소 송금** 리스트
- ⏱ 매트릭스 10초 — 총액/1인 평균, 도넛 차트
- 🖱 한 항목의 **토스로 보내기** 클릭 → 딥링크/QR 등장 → "송금 완료" 체크 → COMPLETED
- ⏱ 송금/완료 10초
- 🎙 "낸 돈을 모아 **최소 횟수로 송금**할 사람을 계산하고, **토스·카카오페이로 바로** 보냅니다."
- ➡ 마무리

### S10. 마무리 — 회고 / 알림 / 아웃트로 (4:50–5:05)
- 🖱 `/notifications` 알림함 1초 — 그동안 쌓인 실시간 알림들
- 🖱 `/groups/:id/recap` 여행 리캡/회고 — 별점·한 줄 후기
- ⏱ 8초
- 🎙 "여행이 끝나면 **회고**까지. 정하고, 떠나고, 깔끔하게 정산하는 그룹 여행. 감사합니다."
- ➡ 타이틀 카드 페이드 아웃(팀명/깃 주소)

## 3. 촬영 순서 팁 (효율)

1. **개별 클립으로** 찍어 편집에서 합치기(한 번에 5분 무편집은 NG 위험 큼).
2. 실시간 장면(S3·S7)은 **두 창을 한 화면에** 두고 한 번에 녹화 — 동기화가 핵심 임팩트.
3. 입력 타이핑은 길면 지루 → **편집에서 1.5~2배속** 또는 자동완성 사용.
4. 외부 API 실패 대비: 검색/길찾기는 **사전에 한 번 호출해 캐시 워밍**(우리 시스템은 DB 캐시) 후 촬영.
5. 마우스 이동을 천천히·또박또박, 클릭 직전 0.3초 정지(시청자 인지 시간).

## 4. 2분 숏컷 버전 (요약 제출/SNS용)

| 시간 | 장면 |
| --- | --- |
| 0:00–0:08 | 인트로 타이틀 |
| 0:08–0:25 | 로그인 + 그룹 생성·초대(실시간 합류) |
| 0:25–0:50 | 장소 검색 + 일정·이동·지도 |
| 0:50–1:20 | **투표 실시간**(두 창) |
| 1:20–1:50 | 정산 최소송금 + 토스 딥링크 |
| 1:50–2:00 | 회고 + 아웃트로 |

> 숏컷은 나레이션 최소화, **실시간 동기화 + 정산 자동화** 두 장면에 집중.

## 5. 나레이션 only 스크립트 (편집 자막/더빙용 모음)

1. (인트로) 친구들과 떠나는 여행, 정하고·짜고·정산하는 걸 한 곳에서.
2. (로그인) 소셜 로그인 한 번이면 끝, 별도 가입이 없습니다.
3. (설문) 5가지 축으로 내 여행 성향을 분석합니다.
4. (초대) 멤버가 들어오면 다른 화면에 1초 안에 반영돼요. 실시간 동기화입니다.
5. (성향매칭) 두 사람 성향을 합쳐 그룹 궁합과 충돌 포인트를 보여줍니다.
6. (장소) 모든 장소는 Google 단일 소스로 검색, 리뷰는 AI가 요약합니다.
7. (일정) 카카오 실측 이동비용이 붙고, 지도로 동선을 한눈에.
8. (투표) 의견이 갈리면 투표로 — 실명·실시간으로 결과가 즉시 반영됩니다.
9. (숙소) 예약가를 넣으면 숙박비가 정산에 자동으로 들어갑니다.
10. (정산) 최소 횟수로 송금할 사람을 계산하고, 토스·카카오페이로 바로 보냅니다.
11. (아웃트로) 정하고, 떠나고, 깔끔하게 정산하는 그룹 여행. 감사합니다.
