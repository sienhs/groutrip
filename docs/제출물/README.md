## 다이어그램 보는 법

`.md` 안의 다이어그램은 **Mermaid** 문법으로 작성되어 있다.

- GitHub/GitLab 웹에서 바로 렌더링됨.
- VS Code: "Markdown Preview Mermaid Support" 확장.
- Word/PDF 변환: <https://mermaid.live> 에 코드 붙여넣기 → PNG/SVG 내보내기 → 문서에 삽입.

## Word/PDF 제출 변환 팁

```bash
# Pandoc 사용 시 (예시)
pandoc 설계문서/01_요구사항정의서.md -o 요구사항정의서.pdf
```

또는 각 `.md`를 VS Code/Typora에서 열어 PDF로 인쇄(Export)한다.
