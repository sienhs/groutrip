## 다시 추출하는 법 (다이어그램 수정 후)

```bash
# 1) 설계문서의 ```mermaid 블록을 .mmd로 추출 → 2) mmdc로 PNG 렌더
#    (Node/npm 필요. 최초 1회 chromium 자동 다운로드)
npx -y @mermaid-js/mermaid-cli -i 입력.mmd -o 출력.png -b white -s 2
```

> 폰트가 깨지면 mermaid-config로 `fontFamily: "Malgun Gothic, Noto Sans KR"` 지정.
